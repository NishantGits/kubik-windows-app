# Kubik - Quick Start Guide

This folder contains a portable, standalone version of the Kubik voxel game engine.

## How to use the app without breaking paths:
1. **IMPORTANT**: Keep the folder structure exactly as it is.
2. The `kubik.exe` executable expects to find the `fonts`, `textures`, `client`, `shaders`, `builtin`, and `games` folders in the SAME directory.
3. Do NOT move `kubik.exe` out of this folder. If you want a desktop shortcut, right-click `kubik.exe` and select "Send to" -> "Desktop (create shortcut)".

## How to Play:
1. Double-click **`kubik.exe`** to start the application.
2. In the Main Menu, select the "Local Game" tab.
3. Choose a "New" world or select an existing one.
4. Select a game (e.g., `devtest`) from the bottom bar.
5. Click **"Play"**!

## Basic Controls:
- **W, A, S, D**: Move
- **Space**: Jump
- **Left Click**: Dig/Punch
- **Right Click**: Place Block/Use
- **I**: Inventory
- **T**: Chat
- **Esc**: Pause/Menu

## Configuration:
You can edit `kubik.conf` with a text editor like Notepad to change settings like graphics, controls, and volume.

Enjoy building in Kubik!
